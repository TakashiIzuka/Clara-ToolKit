# Clara's Supreme Toolkit v1.0 (Universal Edition) 🚀

Toolkit monitoring hardware Windows yang ringan, akurat, dan otomatis. Dirancang khusus untuk memantau suhu Core dan hardware lainnya secara real-time.

### ✨ Fitur Utama:
- **Real-Time CPU Temp**: Menggunakan OpenHardwareMonitorLib untuk akurasi tinggi.
- **Auto-Detect Hardware**: Mendeteksi Brand, Model Laptop, BIOS, RAM, dan SSD secara otomatis.
- **Live Memory & SSD Monitor**: Tampilan visual bar untuk penggunaan RAM dan sisa SSD.
- **Real-Time Clock Speed**: Memantau kecepatan prosesor (Boost Speed).
- **Built-in Cleaner**: Pembersih sampah SSD (Temp & Prefetch).
- **Universal Mode**: Bisa dijalankan di berbagai laptop Windows.

### 🛠️ Cara Penggunaan:
1. Download repository ini.
2. Pastikan file `.bat` dan `.dll` berada di folder yang sama.
3. Klik kanan pada `ClaraSupreme_v27.bat` dan pilih **Run as Administrator**.
4. Selamat memantau!
